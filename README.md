# frappe-bench-base

## Инциализация

```
make up sh
./init.sh
```

Для теста установлен frappe/wiki

## Запуск сервера

```
make start
```

Сайт будет доступен на http://mysite.localhost:8000
Administrator:admin

## Остановить все контейнеры

```
make down
```

## Доступ к контейнеру бэка

```
make sh
```
